# 🚀 SmartCode AI Generator

**SmartCode AI Generator** is an AI-powered web application that turns simple prompts into fully functional **HTML, CSS, and JavaScript websites** in seconds.

Users can describe the type of website they want, and the system instantly generates clean, responsive code along with a live preview.

---

#### ✨ Key Features
- Generate complete websites from natural language prompts  
- Automatic HTML, CSS, and JavaScript code generation  
- Live website preview interface  
- Copy or download the generated code  
- Prompt history using LocalStorage  
- Modern dark-themed UI with smooth animations  
- Fully responsive design  

---


#### 💻 Tech Stack
- HTML5  
- CSS3  
- Vanilla JavaScript  
- LocalStorage API
